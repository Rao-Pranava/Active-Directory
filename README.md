1. Clone this Repository
```

```
2. Move into the ```Active-Directory``` Folder and create a random users for your AD.
```
.\random-users.ps1 Users.json
```
3. Now Add those users created in the ```Json``` file.
```
.\gen-users.ps1 .\Users.json
```
